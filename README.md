# Sturgeon AI Backend

Government Opportunities Intelligence Platform - FastAPI Backend

## Features

- **Opportunity Discovery**: 50,000+ government contracts, grants, SBIR
- **AI Contract Analysis**: GPT-4 powered RFP analysis
- **Automated Compliance**: FAR/DFARS checking
- **Eligibility Scoring**: Smart assessment
- **Grant Management**: Application workflow
- **SBIR/STTR Tools**: Phase tracking
- **Portfolio Management**: Opportunity tracking

## Tech Stack

- FastAPI
- Supabase PostgreSQL
- OpenAI GPT-4
- Vercel deployment
- Python 3.11+
